이 프로젝트는 벡터와 관련된 것을 조사하거나 학습을 해서 발표를 해야한다는 과제 달성을 목표로 만들어졌습니다.
제작 기간: 2017/07/16 22:05~2017/07/17 07:001

LWJGL 만 다루는 것은 처음이니 양해 부탁드립니다.

그래픽 : LWJGL 2.9.3
언어 : JAVA8

이 프로그램은 각 블록의 질량, 탄성계수 등을 고려해서 블록끼리 튕겨져 나가거나 쌓이게 블록을 소환하고 조작할 수 있습니다.

Key_1: 오른쪽으로 힘을 받은 블록을 소환합니다.
Key_2: 오른쪽으로 힘을 받은 블록을 소환합니다.
Key_3: 오른쪽으로 힘을 받은 블록을 소환합니다.
Key_4: 왼쪽으로 힘을 받은 블록을 소환합니다.
Key_5: 왼쪽으로 힘을 받은 블록을 소환합니다.
Key_6: 왼쪽으로 힘을 받은 블록을 소환합니다.
Key_7: 블록 2개를 서로 충돌하게 소환합니다.
Key_Enter: x축으로 힘을 받지 않은 블록을 소환합니다.
Key_BackSpace: 최근 소환한 블록을 삭제합니다.
